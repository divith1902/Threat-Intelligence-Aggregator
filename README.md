# Threat-Intelligence-Aggregator
A Python-based Threat Intelligence Aggregator that parses, normalizes, correlates IOCs and generates security blocklists.
🛡 Threat Intelligence Aggregator (Non-AI)

A Python-based Threat Intelligence Aggregator developed to collect, parse, normalize, and correlate Indicators of Compromise (IOCs) from multiple threat feeds.

This project simulates how Security Operations Centers (SOC) process threat intelligence and generate actionable blocklists for defensive security.

📌 Project Overview

Modern cybersecurity operations rely on threat intelligence feeds from different sources. These feeds often come in different formats and contain duplicate or repeated indicators.

This project:

Collects threat feeds from local files

Extracts IOCs (IP, Domain, URL, Hash)

Normalizes data into a unified structure

Correlates repeated indicators across feeds

Assigns severity levels

Generates blocklists

Produces a final threat summary report

🎯 Key Features

✔ IOC Parsing (IP, Domain, URL, Hash)
✔ Data Normalization
✔ Cross-Feed Correlation Engine
✔ Severity Classification (Low / High)
✔ Blocklist Generation (IP, Domain, URL)
✔ Threat Intelligence Report
